# 融合模型训练说明

## 目标

在已有 `MobileNetV3` 图像分类基线的基础上，引入 `MediaPipe Pose` 骨架特征，训练一个双流融合模型。

## 融合结构

- 图像分支：`MobileNetV3-Small`
- 骨架分支：两层 `MLP`
- 融合方式：分类头前 `concat`

## 骨架特征

当前只使用上半身与躯干关键点：

- 肩：11, 12
- 肘：13, 14
- 腕：15, 16
- 髋：23, 24

每个点保留：

- `x`
- `y`
- `z`
- `visibility`

此外还加入：

- 左上臂向量
- 左前臂向量
- 右上臂向量
- 右前臂向量
- 双肩连线向量
- 若干全局统计量

## 启动前提

先完成骨架提取，确保目录存在：

```text
output/pose_keypoints/
├── train_pose_keypoints.json
├── valid_pose_keypoints.json
└── test_pose_keypoints.json
```

## 启动命令

```bash
python scripts/train_fusion.py --config config/fusion_mobilenetv3_pose.yaml
```

## 输出结果

默认保存到：

```text
experiments/fusion_mobilenetv3_pose/
├── best_model.pth
├── last_model.pth
├── training_log.csv
├── metrics.json
└── class_names.json
```

## 推荐对比

训练完成后，至少对比三组结果：

1. `MobileNetV3 baseline`
2. `MobileNetV3 + Albumentations`
3. `MobileNetV3 + Pose Fusion`
