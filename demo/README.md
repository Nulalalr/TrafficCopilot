# TrafficCopilot Demo

一个基于本地数据集快速搭建的交警手势识别演示原型。

## 方案说明

- 不依赖外部 API。
- 不依赖 PyTorch / OpenCV / scikit-learn。
- 使用 `Pillow + numpy + Flask` 实现：
  - 单帧手势分类：轻量特征 + kNN
  - 时序稳定：滑动窗口 + 简化 FSM
  - 前端展示：上传图片识别 + 预置序列播放

## 启动方式

```bash
python app.py
```

浏览器打开 `http://127.0.0.1:5000`

## 当前限制

- 这是展示型原型，重点在于快速体现系统设计思想，不是高精度量产模型。
- 识别器对数据集分布依赖较强，上传完全不同风格的真实图片时，效果会明显下降。
